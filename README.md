# Sistema de Média Escolar (Visualg)
  ## Descrição
  Sistema desenvolvido em Portugol (Visualg) para calcular a média de um aluno com base em três notas e determinar sua situação final.
  O projeto foi criado com foco na prática de lógica de programação e organização de código
  
  ## Conceitos utilizados
- Variáveis
- Entrada e saída de dados
- EStrutura condicional (se/senão)
- Modularização
- Funções com retorno
- Organização e separação de responsabilidades

  ## Estrutura do projeto
- Calcular a média do aluno
- Definir a situação (Aprovado, Reccuperação ou Reprovado)
- Classificação de notas (A, B, C, D, E, F)

  ## Evolução do projeto
- Versão 1.0 - Cálculo simples da média
- Versão 2.0 - Implementação de classificação por aproveitamento
- Versão 3.0 - Modularização com funções
